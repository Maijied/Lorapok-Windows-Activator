## Method 1 — PowerShell (Automatic)

> [!TIP]
> This is the fastest and easiest method. It downloads and runs the activation script automatically.

### Step 1 — Open PowerShell as Administrator

Press `Win + S`, type **PowerShell**, then click **Run as Administrator**.

<img src="https://github.com/user-attachments/assets/5a10538a-c8c2-4934-868b-fd8e910f1f9e" width="520"/>

---

### Step 2 — Run the Activation Command

Copy and paste the following command into PowerShell, then press **Enter**:

```powershell
irm https://get.activated.win | iex
```

This downloads and launches the activation script automatically.

<img src="https://github.com/user-attachments/assets/1f25dd2a-16db-4053-a45c-aac6f8a9e470" width="520"/>

---

### Step 3 — Select Option 1

In the popup window, press **`1`** to activate Windows permanently. Wait a few seconds for the process to complete.

<img src="https://github.com/user-attachments/assets/0c3689a1-1595-40b3-97e2-041dac423237" width="520"/>

> ✅ **Done!** Your Windows is now activated. Press Enter to close the window.

## Method 2 — CMD with KMS Keys (Manual)

> [!NOTE]
> An active **internet connection** is required for this method.

### Step 1 — Open CMD as Administrator

Press `Win + S`, type **CMD**, then click **Run as Administrator**.

<img src="https://github.com/user-attachments/assets/88ac970a-f24b-4acc-82c0-f8e7c0b05249" width="480"/>

---

### Step 2 — Install the KMS Client Key

Run the command below, replacing `YOUR_LICENSE_KEY` with the key matching your Windows edition from the [table below](#-license-key-reference):

```cmd
slmgr /ipk YOUR_LICENSE_KEY
```

<img src="https://github.com/user-attachments/assets/1b6401e6-7e63-4351-8a92-fa355dfc30cf" width="480"/>

---

### Step 3 — Set the KMS Server Address

```cmd
slmgr /skms kms8.msguides.com
```

<img src="https://github.com/user-attachments/assets/9d634454-1c59-47da-874e-bfe575288fe6" width="480"/>

---

### Step 4 — Activate Windows

```cmd
slmgr /ato
```

<img src="https://github.com/user-attachments/assets/5e495d42-6ce5-4884-9de2-96a56efb4343" width="480"/>

---

### Step 5 — Verify Activation

<img src="https://github.com/user-attachments/assets/9ca70137-8e2e-4504-a52d-22f41d959bd7" width="480"/>

> ✅ **Done!** Your Windows is now permanently activated.
