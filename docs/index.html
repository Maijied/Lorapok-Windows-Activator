ACTIVATION METHODS

Choose Your Method

Two proven approaches — pick whichever suits you best.

⚡ Recommended
🖥️

Method 1 — PowerShell

Fully automatic. One command downloads and runs the activation script for you.

01

Open PowerShell as Administrator

Press Win + S, type PowerShell, then click Run as Administrator.

02

Run the Activation Command

Copy and paste this command, then press Enter:

irm https://get.activated.win | iex
03

Select Option 1

In the popup window, press 1 to activate Windows permanently. Wait a few seconds.

✅ Done! Your Windows is now permanently activated.
⌨️

Method 2 — CMD (Manual)

Manual KMS activation using Command Prompt. Requires an internet connection.

01

Open CMD as Administrator

Press Win + S, type CMD, then click Run as Administrator.

02

Install KMS Client Key

Replace YOUR_KEY with the key matching your Windows edition from the table below.

slmgr /ipk YOUR_KEY
03

Set KMS Server Address

slmgr /skms kms8.msguides.com
04

Activate Windows

slmgr /ato
✅ Done! Your Windows is now permanently activated.
